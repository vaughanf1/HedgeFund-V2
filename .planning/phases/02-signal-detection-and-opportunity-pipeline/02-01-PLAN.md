**volume_spike.py** (SGNL-01):
- `from __future__ import annotations`
- Read `VOLUME_ZSCORE_THRESHOLD` from env (default "2.0")
- `VOLUME_SPIKE_SQL` as `text(...)`: CTE `recent` computes AVG and STDDEV_SAMP over 20 PRECEDING rows (partitioned by ticker, ordered by timestamp, WHERE ticker = :ticker AND timestamp >= NOW() - INTERVAL '30 days'). CTE `latest` takes top row and computes z_score = `(volume - avg_vol_20d) / NULLIF(std_vol_20d, 0)`. Outer SELECT filters WHERE z_score >= :threshold.
- `detect_volume_spike(session, ticker: str) -> dict | None`: execute SQL with params, return `{"signal_type": "volume_spike", "ticker": ..., "score": min(z_score / 3.0, 1.0), "detail": {...}}` or None.
- Handle: if row is None or z_score is None, return None.

**price_breakout.py** (SGNL-02):
- `from __future__ import annotations`
- Read `GAP_THRESHOLD` from env (default "0.03")
- `PRICE_BREAKOUT_SQL` as `text(...)`: CTE `windowed` computes MAX(close) and MIN(close) over 20 PRECEDING rows + LAG(close). Outer SELECT determines breakout_type: "breakout_up" if close > high_20d, "breakout_down" if close < low_20d, "gap_up" if pct change > threshold, "gap_down" if pct change < -threshold, else NULL. LIMIT 1.
- `detect_price_breakout(session, ticker: str) -> dict | None`: execute SQL, if row is None or breakout_type is None, return None. Score: 0.8 for breakout_up/breakout_down, 0.6 for gap_up/gap_down. Return signal dict with detail including breakout_type, close, high_20d, low_20d.

**sector_momentum.py** (SGNL-05):
- `from __future__ import annotations`
- Read `SECTOR_MAP` from env as JSON (default `'{}'`), parse with `json.loads()`
- Read `SECTOR_MOMENTUM_THRESHOLD` from env (default "0.02")
- Read `SECTOR_MIN_COVERAGE` from env (default "0.6")
- `RETURN_5D_SQL` as `text(...)`: ROW_NUMBER OVER (PARTITION BY ticker ORDER BY timestamp DESC), filter rn IN (1, 6), compute 5-day return per ticker for all tickers in `ANY(:tickers)`, timestamp >= NOW() - INTERVAL '10 days'.
- `detect_sector_momentum(session, ticker: str, watchlist: list[str]) -> dict | None`:
  1. Look up ticker's sector in SECTOR_MAP. If not in map, return None.
  2. Find sector_peers = [t for t in watchlist if SECTOR_MAP.get(t) == sector].
  3. Execute RETURN_5D_SQL with sector_peers as :tickers param.
  4. If fewer than `SECTOR_MIN_COVERAGE * len(sector_peers)` tickers returned data, return None.
  5. Compute median return across sector peers. If ticker's return exceeds median by >= threshold, return signal dict with score = min((ticker_return - median) / 0.05, 1.0). Detail includes ticker_return, sector_median, sector name.
