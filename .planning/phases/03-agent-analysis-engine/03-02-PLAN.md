**`_run_graph_sync(persona_name, data_context, redis_url)` helper:**

**`run_persona_agent` Celery task:**
  - `@app.task(name="app.tasks.analyse_opportunity.run_persona_agent", bind=True, max_retries=3)`
  - Signature: `def run_persona_agent(self, opportunity_id: str, persona_name: str, data_context: dict) -> None:`
  - Create sync redis client: `r = redis.from_url(_REDIS_URL)`.
  - Publish AGENT_STARTED event: `publish_event(r, "AGENT_STARTED", {"opportunity_id": opportunity_id, "persona": persona_name})`.
  - Try: `verdict_dict = _run_graph_sync(persona_name, data_context, _REDIS_URL)`.
  - Except Exception as exc: `self.retry(exc=exc, countdown=30)`.
  - Persist verdict to Redis hash: `r.hset(f"verdicts:{opportunity_id}", persona_name, json.dumps(verdict_dict, default=str))`.
  - Set TTL on verdicts hash: `r.expire(f"verdicts:{opportunity_id}", VERDICT_TTL)`.
  - Atomic counter: `completed = int(r.hincrby(f"verdicts_counter:{opportunity_id}", "count", 1))`.
  - Set TTL on counter: `r.expire(f"verdicts_counter:{opportunity_id}", VERDICT_TTL)`.
  - Publish AGENT_COMPLETE: `publish_event(r, "AGENT_COMPLETE", {"opportunity_id": opportunity_id, "persona": persona_name, "verdict": verdict_dict.get("verdict"), "confidence": verdict_dict.get("confidence")})`.
  - Log: completed count.
  - If `completed >= AGENT_COUNT`: `run_committee.delay(opportunity_id)`.
  - Close redis client: `r.close()`.

**`fan_out` Celery task:**
  - `@app.task(name="app.tasks.analyse_opportunity.fan_out", bind=True)`
  - Signature: `def fan_out(self, opportunity: dict) -> None:`
  - Build `opportunity_id = f"{opportunity['ticker']}:{opportunity['detected_at']}"`.
  - `data_context = opportunity` (the full opportunity dict from Phase 2 queue; partitioner inside graph node handles filtering).
  - Log: "Fanning out to {AGENT_COUNT} agents for {opportunity_id}".
  - For each persona in AGENTS: `run_persona_agent.delay(opportunity_id, persona, data_context)`.

**`consume_queue` Celery task:**
  - `@app.task(name="app.tasks.analyse_opportunity.consume_queue", bind=True)`
  - Signature: `def consume_queue(self) -> None:`
  - Create sync redis client.
  - `while True:`:
    - `result = r.blpop(OPP_QUEUE_KEY, timeout=30)`.
    - If None: continue (timeout, loop back).
    - `_, raw = result`.
    - `opportunity = json.loads(raw)`.
    - Log: "Dequeued opportunity: {opportunity.get('ticker')}".
    - `fan_out.delay(opportunity)`.
  - This is a long-running task — it blocks on BLPOP and loops forever. It should be started as a dedicated Celery worker or via `consume_queue.delay()` at startup.

**variance.py:**
- `from __future__ import annotations`
- Import statistics, os, logging.
- Import AgentVerdict from app.agents.schemas.
- `MINIMUM_VARIANCE_THRESHOLD = float(os.environ.get("AGENT_VARIANCE_THRESHOLD", "8.0"))`.
- `logger = logging.getLogger(__name__)`.
- `def compute_variance_score(verdicts: list[AgentVerdict]) -> float:`:
  - `scores = [v.confidence for v in verdicts]`
  - If `len(scores) < 2`: return 0.0.
  - Return `statistics.stdev(scores)`.
- `def is_committee_valid(verdicts: list[AgentVerdict]) -> bool:`:
  - `variance = compute_variance_score(verdicts)`.
  - `logger.info("Inter-agent variance score: %.2f (threshold: %.2f)", variance, MINIMUM_VARIANCE_THRESHOLD)`.
  - Return `variance >= MINIMUM_VARIANCE_THRESHOLD`.
