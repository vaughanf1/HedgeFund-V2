**asymmetric.py (ASYM-01, ASYM-02, ASYM-03):**
- `from __future__ import annotations`
- Import logging; AgentVerdict from app.agents.schemas.
- `logger = logging.getLogger(__name__)`
- Define `def evaluate_asymmetric(verdicts: list[AgentVerdict], opportunity: dict) -> dict:`:
  - Purpose: Identify if this opportunity has 5x-10x asymmetric upside potential.
  - Logic:
    - Count `buy_count` = number of verdicts with verdict == "BUY".
    - Compute `avg_confidence` = mean of confidence scores for BUY verdicts (0 if none).
    - Collect all `upside_scenario` strings from BUY verdicts.
    - Collect all `risks` from all verdicts (flattened).
    - `is_asymmetric = buy_count >= 3 and avg_confidence >= 70`.
    - If `is_asymmetric`:
      - `catalyst_justification` = "; ".join of upside_scenario strings from BUY verdicts.
      - `probability_score` = avg_confidence / 100.0.
      - `payoff_multiple` = "5x-10x" if avg_confidence >= 80 else "3x-5x".
      - `required_conditions` = upside_scenario strings as list.
      - `risk_flags` = unique risks across all verdicts.
    - Else: catalyst_justification = None, probability_score = 0.0, payoff_multiple = "1x", required_conditions = [], risk_flags = risks.
  - Return dict: `{"is_asymmetric": is_asymmetric, "catalyst_justification": catalyst_justification, "probability_score": probability_score, "payoff_multiple": payoff_multiple, "required_conditions": required_conditions, "risk_flags": risk_flags}`.
  - Log result.

**committee.py (CIO-01, CIO-02, CIO-03):**
- `from __future__ import annotations`
- Import logging; AgentVerdict, CommitteeReport from app.agents.schemas.
- `logger = logging.getLogger(__name__)`
- Define `REGIME_WEIGHTS` dict as specified in research:
  ```
  "macro":       {"buffett": 0.7, "munger": 0.8, "ackman": 0.9, "cohen": 1.0, "dalio": 1.5}
  "fundamental": {"buffett": 1.5, "munger": 1.4, "ackman": 1.2, "cohen": 0.8, "dalio": 0.9}
  "momentum":    {"buffett": 0.7, "munger": 0.8, "ackman": 1.0, "cohen": 1.5, "dalio": 1.2}
  "default":     {"buffett": 1.0, "munger": 1.0, "ackman": 1.0, "cohen": 1.0, "dalio": 1.0}
  ```
- Define `def detect_regime(opportunity: dict) -> str:`:
  - Extract `signals = opportunity.get("signals", [])`.
  - `signal_types = {s.get("signal_type") for s in signals if isinstance(s, dict) and s.get("score", 0) > 0.5}` (handle score as float or str).
  - If "sector_momentum" or "price_breakout" in signal_types: return "momentum".
  - If "insider_cluster" in signal_types: return "fundamental".
  - If "macro" in any signal type string: return "macro".
  - Return "default".
- Define `def aggregate_committee(opportunity_id: str, verdicts: list[AgentVerdict], opportunity: dict, asymmetric_result: dict) -> CommitteeReport:`:
  - `regime = detect_regime(opportunity)`.
  - `weights = REGIME_WEIGHTS.get(regime, REGIME_WEIGHTS["default"])`.
  - Compute `weighted_conviction`:
    - `total_weight = sum(weights.get(v.persona, 1.0) for v in verdicts)`.
    - `weighted_sum = sum(v.confidence * weights.get(v.persona, 1.0) for v in verdicts)`.
    - `weighted_conviction = weighted_sum / total_weight if total_weight > 0 else 0.0`.
  - Determine consensus:
    - Count verdicts by verdict value (BUY/HOLD/PASS).
    - If any verdict has >= 3 votes: consensus = that verdict.
    - Else: consensus = "SPLIT".
  - `dissent_agents` = personas whose verdict != consensus (empty list if SPLIT).
  - Import `compute_variance_score` from `app.analysis.variance`.
  - `variance_score = compute_variance_score(verdicts)`.
  - Build and return `CommitteeReport(opportunity_id=opportunity_id, verdicts=verdicts, consensus=consensus, dissent_agents=dissent_agents, variance_score=variance_score, weighted_conviction=weighted_conviction, asymmetric_flag=asymmetric_result["is_asymmetric"], asymmetric_justification=asymmetric_result.get("catalyst_justification"))`.
  - Log: regime, consensus, weighted_conviction, variance_score.

**cio.py (CIO-04, CIO-05):**
- `from __future__ import annotations`
- Import logging; CommitteeReport, CIODecision from app.agents.schemas.
- `logger = logging.getLogger(__name__)`
- Define `def make_cio_decision(report: CommitteeReport) -> CIODecision:`:
  - Deterministic derivation from CommitteeReport fields:
  - `conviction_score = int(round(report.weighted_conviction))`.
  - Allocation tiers (based on conviction):
    - conviction >= 80: `allocation = 8.0`
    - conviction >= 65: `allocation = 5.0`
    - conviction >= 50: `allocation = 3.0`
    - conviction >= 35: `allocation = 1.5`
    - else: `allocation = 0.0`
  - If `report.asymmetric_flag` and conviction >= 50: multiply allocation by 1.5 (capped at 10.0).
  - Risk rating:
    - variance_score > 25: "VERY_HIGH" (high disagreement = high uncertainty)
    - variance_score > 15: "HIGH"
    - variance_score > 8: "MEDIUM"
    - else: "LOW"
  - Time horizon: mode of verdict time_horizons across the five agents (most common string). If tie, pick the first alphabetically.
  - Key catalysts: collect unique upside_scenario values from BUY verdicts (up to 5).
  - Kill conditions: collect unique risks from all verdicts where confidence < 40 (up to 5). If none, use top 3 risks from lowest-confidence agent.
  - Final verdict:
    - If consensus is "BUY" and conviction >= 50: "INVEST"
    - If consensus is "SPLIT" and conviction >= 60: "MONITOR"
    - If consensus is "HOLD": "MONITOR"
    - Else: "PASS"
  - Build and return `CIODecision(opportunity_id=report.opportunity_id, conviction_score=conviction_score, suggested_allocation_pct=allocation, time_horizon=time_horizon, risk_rating=risk_rating, key_catalysts=key_catalysts, kill_conditions=kill_conditions, final_verdict=final_verdict)`.
  - Log: final_verdict, conviction_score, allocation.
