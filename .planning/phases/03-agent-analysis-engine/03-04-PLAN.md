**events.py:**
- `from __future__ import annotations`
- Imports: asyncio, os, logging; APIRouter from fastapi; redis.asyncio as aioredis; EventSourceResponse from sse_starlette.sse.
- `router = APIRouter(prefix="/api/v1/events", tags=["events"])`
- `_REDIS_URL = os.environ.get("REDIS_URL", "redis://redis:6379/0")`
- `logger = logging.getLogger(__name__)`

- `@router.get("/stream")`:
  - `async def stream_pipeline_events():`:
    - Define `async def event_generator():`:
      - `r = aioredis.from_url(_REDIS_URL)`.
      - `pubsub = r.pubsub()`.
      - `await pubsub.subscribe("pipeline:events")`.
      - `try:`:
        - `async for message in pubsub.listen():`:
          - `if message["type"] == "message":`:
            - `data = message["data"]`
            - If data is bytes: `data = data.decode("utf-8")`.
            - `yield {"event": "pipeline", "data": data}`.
      - `finally:`:
        - `await pubsub.unsubscribe("pipeline:events")`.
        - `await r.aclose()`.
    - `return EventSourceResponse(event_generator())`.

**opportunities.py:**
- `from __future__ import annotations`
- Imports: json, logging; APIRouter, Depends, Query, HTTPException from fastapi; AsyncSession from sqlalchemy.ext.asyncio; select, desc from sqlalchemy; get_session from app.db.deps; CIODecisionRecord, AgentVerdictRecord from app.db.models; BaseModel from pydantic; Optional, Any from typing.
- `router = APIRouter(prefix="/api/v1/opportunities", tags=["opportunities"])`
- `logger = logging.getLogger(__name__)`

- Define response models:
  - `class OpportunitySummary(BaseModel):`:
    - `opportunity_id: str`
    - `conviction_score: int`
    - `suggested_allocation_pct: float`
    - `final_verdict: str`
    - `risk_rating: str` (extract from decision_json)
    - `decided_at: str`
    - `model_config = {"from_attributes": True}`
  - `class OpportunityDetail(BaseModel):`:
    - `opportunity_id: str`
    - `decision: dict[str, Any]` (full CIODecision JSON)
    - `verdicts: list[dict[str, Any]]` (full AgentVerdict JSONs)

- `@router.get("", response_model=list[OpportunitySummary])`:
  - `async def list_opportunities(limit: int = Query(20, le=100), final_verdict: Optional[str] = Query(None), session: AsyncSession = Depends(get_session)):`:
  - Query `CIODecisionRecord` ordered by `conviction_score` desc, limit.
  - If `final_verdict` filter provided: add where clause.
  - Execute query. Map results to `OpportunitySummary`:
    - Extract risk_rating from decision_json (parse JSON, get "risk_rating" key).
    - `decided_at` = record.decided_at.isoformat().
  - Return list.

- `@router.get("/{opportunity_id}")`:
  - `async def get_opportunity_detail(opportunity_id: str, session: AsyncSession = Depends(get_session)):`:
  - Query `CIODecisionRecord` where opportunity_id matches. If not found: raise HTTPException(404).
  - Parse `decision_json` to dict.
  - Query `AgentVerdictRecord` where opportunity_id matches, ordered by persona.
  - Parse each `verdict_json` to dict.
  - Return `OpportunityDetail(opportunity_id=opportunity_id, decision=decision_dict, verdicts=verdict_dicts)`.
